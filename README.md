# How to Run the Notebook in Google Colab
This project is designed to be executed using **Google Colab** to simplify setup and leverage cloud-based computing resources.

### Step 1 — Open Google Colab
1. Go to: https://colab.research.google.com
2. Click **Upload**
3. Upload the provided `.ipynb` notebook file

### Step 2 — Run the Notebook

Click Runtime → Run all,
The notebook will execute the full pipeline (training, evaluation, and visualization)

# Why KaggleHub Is Used

1. The dataset size is relatively large and not suitable for direct inclusion in a GitHub repository.
2. KaggleHub enables automatic dataset downloading from an official source.
3. This approach improves reproducibility and follows common machine learning best practices.
